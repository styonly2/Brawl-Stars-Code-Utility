**Brawl Stars Code Utility** — это легкий и быстрый инструмент на Java для работы с идентификаторами (ID) и игровыми тегами команд. Проект создан для исследователей механик игры и разработчиков модов.

---

## 🇷🇺 Описание (RU)

### ✨ Основные функции:
* **Генерация кодов**: Создание случайных игровых тегов на основе математических алгоритмов.
* **Конвертация ID → ТЕГ**: Узнайте, какой буквенный код соответствует конкретному числовому ID.
* **Конвертация ТЕГ → ID**: Получите точный порядковый номер любого игрового тега.
* **Стабильность**: Полностью переработанная система ввода, исключающая лаги в консоли Windows.
* **Интерфейс**: Цветная ANSI-графика для удобной навигации в меню.

### 🚀 Инструкция по запуску:
1. Установите [JDK 17](https://www.oracle.com/java/technologies/javase-jdk17-downloads.html) или выше.
2. Скачайте файлы проекта в одну папку.
3. Запустите файл `run.bat`или файл .exe.
